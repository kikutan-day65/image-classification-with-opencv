# Procedure
Before you use this code, download the zip file containing pdfs [here](https://drive.google.com/file/d/1Y0SvP-S8nrEcesTun4FV5tJGr9Hsk73z/view)
Do NOT change the zip file name you downloaded!

## Steps
1. Import libraries with ```pip install -r requirements.txt```
2. Unzip the pdf file with ```unzip fileName.zip```
2. Execute ```python3 exercise8-2_mori.py``` to create directories
3. Execute ```python3 exercise8-1_mori.py``` to classify images
4. You can find the result in the result directory
